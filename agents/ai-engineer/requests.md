## How to Request Work
Add a new entry below with:
- Requesting agent
- Task description
- Priority (High/Medium/Low)
- Deadline (if any)

## Model Selection Guide
When requesting AI work, specify:
- **Speed priority**: Use Ollama (local, fast)
- **Quality priority**: Use Kimi (cloud, free)
- **Reasoning required**: Use MiniMax
- **Web search needed**: Use Grok
