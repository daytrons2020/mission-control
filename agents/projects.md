# Mission Control Platform - Project Overview

## Vision
A multi-agent system for managing Discord bots, trading automation, and AI workflows through coordinated specialist agents.

## Modules

### Core
- Discord bot management
- Agent orchestration
- Cron job scheduling
- Memory and state management

### Trading (Future)
- Market data analysis
- Trade execution
- Portfolio tracking
- Risk management

### Cost Tracking
- Token usage monitoring
- API cost analysis
- Budget alerts

## Agent Roster

| Agent | Role | Status |
|-------|------|--------|
| Integration Specialist | External services, APIs, webhooks | ✅ Active |
| Frontend Developer | UI components, dashboards | ✅ Active |
| Database Engineer | Data persistence, queries | ✅ Active |
| Backend Developer | API logic, orchestration | ✅ Active |
| AI Engineer | Models, prompts, optimization | ✅ Active |

## Tech Stack
- **Orchestration**: OpenClaw
- **Primary Model**: Kimi K2.5 (cloud, free)
- **Local Model**: Ollama qwen3:8b
- **Messaging**: Discord, NATS (planned)
- **Storage**: File-based (Markdown, JSON)
