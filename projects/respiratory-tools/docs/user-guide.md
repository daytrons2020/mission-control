# Respiratory Tools - User Guide

## Quick Start

### For Staff (Mobile App)

#### 1. First Time Setup
1. Open the app URL in your phone's browser
2. Tap "Add to Home Screen" when prompted
3. Launch the app from your home screen
4. Select your name from the staff list
5. Enter your 4-digit PIN

#### 2. Submitting Availability
1. Tap "My Availability" on the home screen
2. Select the week you want to submit for
3. Tap each day to mark your availability:
   - **Green** = Available for that shift
   - **Red** = Unavailable
   - **Yellow** = Preferred shift
4. Add any notes (optional)
5. Tap "Submit Availability"

#### 3. Viewing Your Schedule
1. Tap "My Schedule" on the home screen
2. View your assigned shifts
3. Tap a shift for details
4. Request swaps or time off from shift details

#### 4. Scanning Inventory
1. Tap "Inventory Scanner" on the home screen
2. Point camera at item QR code
3. Select action:
   - **+** Add stock (receiving)
   - **-** Remove stock (usage)
   - **=** Physical count
   - **🛒** Add to order list
4. Enter quantity
5. Tap "Confirm"

---

### For Managers (Desktop App)

#### 1. Opening the Dashboard
1. Open `Respiratory_Tools/Admin_Dashboard.exe`
2. Or open `index.html` in a web browser

#### 2. Reviewing Availability
1. Click "Availability" tab
2. View all staff submissions
3. Click "Generate Schedule" to run auto-balancer
4. Review and adjust assignments
5. Click "Publish" to notify staff

#### 3. Managing Inventory
1. Click "Inventory" tab
2. View current stock levels
3. Review scan history
4. Generate order list from "Order Queue"
5. Export to Excel for purchasing

#### 4. Syncing Data
1. Connect mobile devices via USB or shared folder
2. Click "Sync" button
3. Review sync results
4. Resolve any conflicts

## Troubleshooting

### Mobile App Won't Sync
- Check that you're connected to WiFi
- Try manual sync from Settings
- Contact IT if sync folder is inaccessible

### QR Code Won't Scan
- Ensure good lighting
- Hold phone steady
- Clean camera lens
- Try entering Lawson number manually

### Schedule Looks Wrong
- Check availability submissions are current
- Verify shift templates are correct
- Review auto-balancer constraints
- Manually adjust if needed

## FAQ

**Q: Can I use this on my personal phone?**
A: Yes, the PWA works on any smartphone with a modern browser.

**Q: What if I don't have internet?**
A: The app works offline. Data syncs when you reconnect.

**Q: How do I change my PIN?**
A: Ask your manager to reset it from the admin dashboard.

**Q: Can I swap shifts with a coworker?**
A: Yes! Request a swap from your schedule view. Your manager will approve.
