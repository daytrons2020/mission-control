- **Search Tips:**
  - MeSH terms: "Respiratory Therapy," "Mechanical Ventilation," "Respiratory Care"
  - Filters: Clinical Trial, Systematic Review, Guideline

- Strong respiratory care content

- **URL:** https://www.cochranelibrary.com
