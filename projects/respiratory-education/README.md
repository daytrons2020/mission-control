# Respiratory Education Content System

## Project Overview
This repository contains educational content systems for respiratory therapy education, including slide deck templates, workshop frameworks, skills lab scenarios, and resource collections.

## Directory Structure

```
respiratory-education/
├── templates/          # Slide deck templates (markdown → PowerPoint workflow)
├── workshops/          # Workshop planning templates
├── content-library/    # Content outline for respiratory topics
├── skills-labs/        # Skills lab scenarios
└── resources/          # Resource collection (links, references, standards)
```

## Quick Start

1. **Slide Decks**: See `templates/` for markdown-based presentation templates
2. **Workshops**: See `workshops/` for planning frameworks
3. **Skills Labs**: See `skills-labs/` for simulation scenarios
4. **Resources**: See `resources/` for standards and references

## Standards & Guidelines

This project aligns with:
- American Association for Respiratory Care (AARC) Clinical Practice Guidelines
- National Board for Respiratory Care (NBRC) educational standards
- Simulation-based education best practices
