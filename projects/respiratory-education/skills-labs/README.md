# Skills Lab Scenarios

## Overview

This directory contains simulation scenarios and skills lab exercises for respiratory therapy education. Scenarios are organized by complexity level and clinical domain.

## Scenario Categories

### Basic Scenarios (Entry-Level)
1. Oxygen Therapy Administration
2. Aerosol Medication Delivery
3. Chest Physiotherapy
4. Basic Airway Management
5. Arterial Blood Gas Collection

### Intermediate Scenarios (Staff Level)
1. Noninvasive Ventilation Initiation
2. Mechanical Ventilator Setup
3. Endotracheal Intubation Assistance
4. Emergency Airway Management
5. Ventilator Weaning

### Advanced Scenarios (Critical Care)
1. ARDS Management
2. Severe Asthma Exacerbation
3. Difficult Airway Management
4. ECMO Patient Management
5. Transport Ventilation

## Scenario Format

Each scenario follows a standardized format:

```
[Scenario Number]: [Title]
├── Patient Profile
├── Learning Objectives
├── Equipment Needed
├── Scenario Flow
├── Expected Actions
├── Debriefing Points
└── References
```

## BASIC SCENARIOS

### Scenario 1.1: Oxygen Therapy Selection and Administration

**Level:** Basic  
**Duration:** 30 minutes  
**Setting:** General Floor / Skills Lab

#### Patient Profile
- **Name:** John Miller
- **Age:** 68
- **Gender:** Male
- **Diagnosis:** COPD exacerbation, community-acquired pneumonia
- **Background:** Admitted 2 days ago, currently on 2L NC, complaining of increased shortness of breath

#### Vital Signs
- HR: 98 bpm
- BP: 142/88 mmHg
- RR: 24/min
- SpO2: 88% on 2L NC
- Temp: 37.8°C

#### Learning Objectives
1. Select appropriate oxygen delivery device based on patient needs
2. Calculate appropriate FIO2
3. Monitor for complications of oxygen therapy
4. Document appropriately

#### Equipment Needed
- Nasal cannula (various sizes)
- Simple mask
- Venturi mask with color-coded valves
- Non-rebreather mask
- High-flow nasal cannula setup
- Pulse oximeter
- Stethoscope

#### Scenario Flow

**Phase 1: Assessment (5 min)**
- Participant enters room
- Patient reports increased dyspnea
- Current SpO2: 88% on 2L NC

**Phase 2: Intervention (10 min)**
- Participant assesses patient
- Selects appropriate device
- Applies oxygen therapy
- Monitors response

**Phase 3: Reassessment (10 min)**
- SpO2 improves to 92% on new device
- Patient reports decreased dyspnea
- Participant documents and plans follow-up

**Phase 4: Complication (5 min)**
- Patient with COPD: monitor for CO2 retention
- Participant recognizes need for ABG

#### Expected Actions
1. Perform focused respiratory assessment
2. Increase oxygen flow appropriately OR change device
3. Monitor SpO2 continuously
4. Assess work of breathing
5. Notify physician of changes
6. Document assessment and interventions

#### Critical Errors
- [ ] Applying NRB to COPD patient without orders
- [ ] Failing to assess before intervention
- [ ] Not monitoring response

#### Debriefing Points
1. Device selection rationale
2. FIO2 calculations
3. COPD considerations (CO2 retention risk)
4. Documentation requirements

### Scenario 1.2: Metered-Dose Inhaler Technique Training

**Level:** Basic  
**Duration:** 20 minutes  
**Setting:** Outpatient / Skills Lab

#### Patient Profile
- **Name:** Sarah Chen
- **Age:** 34
- **Gender:** Female
- **Diagnosis:** Asthma
- **Background:** Newly diagnosed, first prescription for albuterol MDI

#### Learning Objectives
1. Demonstrate proper MDI technique
2. Teach technique to patient effectively
3. Assess patient understanding (teach-back method)
4. Document patient education

#### Equipment Needed
- Placebo MDI
- Spacer devices (various types)
- Patient education materials
- Checklist for technique steps

#### Scenario Flow

**Phase 1: Assessment (5 min)**
- Participant reviews medication order
- Assesses patient's baseline knowledge
- Checks for physical limitations

**Phase 2: Demonstration (5 min)**
- Participant demonstrates proper technique
- Explains each step clearly

**Phase 3: Patient Practice (5 min)**
- Patient attempts technique
- Participant provides feedback
- Corrects errors

**Phase 4: Teach-Back (5 min)**
- Patient demonstrates technique
- Participant assesses understanding
- Documents education

#### Expected Actions
1. Verify medication order
2. Assess patient's ability to use device
3. Demonstrate proper technique
4. Have patient demonstrate back
5. Provide written materials
6. Document education provided

#### MDI Technique Checklist
- [ ] Remove cap and shake inhaler
- [ ] Exhale fully
- [ ] Position inhaler (1-2 inches from mouth OR with spacer)
- [ ] Start breathing in slowly
- [ ] Press canister while inhaling
- [ ] Continue slow deep breath
- [ ] Hold breath 10 seconds
- [ ] Wait 30-60 seconds between puffs

#### Debriefing Points
1. Common patient errors
2. Spacer benefits and use
3. Health literacy considerations
4. Documentation of patient education

### Scenario 1.3: Chest Physiotherapy and Postural Drainage

**Level:** Basic  
**Duration:** 30 minutes  
**Setting:** Inpatient / Skills Lab

#### Patient Profile
- **Name:** Robert Johnson
- **Age:** 72
- **Gender:** Male
- **Diagnosis:** Bronchiectasis with retained secretions
- **Background:** Productive cough, thick secretions, needs airway clearance

#### Learning Objectives
1. Position patient for postural drainage
2. Perform percussion and vibration techniques
3. Coach effective coughing
4. Monitor patient tolerance

#### Equipment Needed
- Pillows for positioning
- Percussion cup or cupped hands
- Suction equipment (available)
- Emesis basin
- Tissues
- Pulse oximeter

#### Scenario Flow

**Phase 1: Assessment (5 min)**
- Review chest X-ray (bilateral lower lobe infiltrates)
- Auscultate chest
- Assess patient's ability to tolerate procedure

**Phase 2: Positioning (5 min)**
- Position for lower lobes (trendelenburg with pillows)
- Ensure patient comfort and safety

**Phase 3: Treatment (15 min)**
- Percussion to targeted areas
- Vibration during exhalation
- Coach coughing between positions
- Monitor SpO2 and comfort

**Phase 4: Post-Treatment (5 min)**
- Return to comfortable position
- Auscultate post-treatment
- Document procedure and response

#### Expected Actions
1. Review orders and patient history
2. Explain procedure to patient
3. Position correctly for target areas
4. Perform percussion with appropriate technique
5. Apply vibration correctly
6. Coach effective cough
7. Monitor patient throughout
8. Document procedure

#### Critical Errors
- [ ] Percussing over spine, sternum, or surgical areas
- [ ] Performing on full stomach
- [ ] Continuing despite patient distress
- [ ] Improper body mechanics (participant injury risk)

#### Debriefing Points
1. Positioning for different lung segments
2. Contraindications to CPT
3. Patient tolerance assessment
4. Alternative airway clearance techniques

### Scenario 2.1: Noninvasive Ventilation Initiation

**Level:** Intermediate  
**Duration:** 45 minutes  
**Setting:** Emergency Department / ICU

#### Patient Profile
- **Name:** Maria Garcia
- **Age:** 56
- **Gender:** Female
- **Diagnosis:** Acute COPD exacerbation, acute-on-chronic respiratory failure
- **Background:** Increased work of breathing, not responding to supplemental O2

#### Vital Signs
- HR: 118 bpm
- BP: 156/92 mmHg
- RR: 32/min, shallow
- SpO2: 86% on 6L NC
- ABG: pH 7.28, PaCO2 68, PaO2 58, HCO3 32

#### Learning Objectives
1. Recognize indications for NIV
2. Select appropriate interface
3. Initiate BiPAP with appropriate settings
4. Troubleshoot common issues
5. Monitor patient response

#### Equipment Needed
- BiPAP machine
- Full-face masks (multiple sizes)
- Nasal masks (multiple sizes)
- Headgear
- Humidifier
- Pulse oximeter
- Capnography (optional)
- Suction equipment

#### Scenario Flow

**Phase 1: Assessment (10 min)**
- Participant reviews ABG and clinical picture
- Determines NIV is indicated
- Explains procedure to patient
- Selects interface

**Phase 2: Initiation (15 min)**
- Fit mask appropriately
- Start with low pressures (IPAP 10, EPAP 5)
- Gradually titrate up
- Coach patient on breathing with machine

**Phase 3: Troubleshooting (10 min)**
- Patient reports mask discomfort
- Leak alarm sounds
- Participant adjusts fit and settings

**Phase 4: Monitoring (10 min)**
- SpO2 improves to 92%
- RR decreases to 22/min
- Patient more comfortable
- Document settings and response

#### Expected Actions
1. Verify NIV indication
2. Rule out contraindications
3. Select appropriate interface
4. Start with appropriate settings
5. Titrate based on response
6. Troubleshoot leaks and comfort issues
7. Monitor for complications
8. Document thoroughly

#### Contraindications to Check
- Respiratory arrest
- Hemodynamic instability
- Inability to protect airway
- Uncooperative patient
- Facial trauma/surgery
- Copious secretions

#### Debriefing Points
1. NIV indications and contraindications
2. Interface selection criteria
3. Initial settings and titration
4. Monitoring parameters
5. When to escalate to intubation

### Scenario 2.2: Mechanical Ventilator Setup and Initial Management

**Level:** Intermediate  
**Duration:** 45 minutes  
**Setting:** ICU / Skills Lab

#### Patient Profile
- **Name:** David Williams
- **Age:** 45
- **Gender:** Male
- **Diagnosis:** Post-operative (abdominal surgery), respiratory failure
- **Background:** Intubated in OR, arriving to ICU on ventilator

#### Learning Objectives
1. Perform ventilator safety check
2. Set up ventilator with appropriate initial settings
3. Verify patient-ventilator synchrony
4. Set appropriate alarms
5. Document ventilator settings

#### Equipment Needed
- Mechanical ventilator
- Ventilator circuit
- Humidifier
- Suction equipment
- Stethoscope
- Pulse oximeter
- Capnography
- Bag-valve-mask

#### Scenario Flow

**Phase 1: Safety Check (10 min)**
- Participant performs full ventilator check
- Tests alarms
- Verifies oxygen source
- Checks circuit integrity

**Phase 2: Initial Setup (15 min)**
- Calculate ideal body weight
- Select mode (Volume A/C)
- Set tidal volume (6-8 ml/kg IBW)
- Set rate (12-16)
- Set PEEP (5 cmH2O)
- Set FIO2 (100% initially)

**Phase 3: Patient Connection (10 min)**
- Connect to patient
- Assess synchrony
- Check exhaled volumes
- Auscultate breath sounds

**Phase 4: Alarm Setting (10 min)**
- Set high/low pressure alarms
- Set low minute ventilation alarm
- Set high respiratory rate alarm
- Verify all alarms functional

#### Expected Actions
1. Complete pre-use safety check
2. Calculate appropriate settings based on IBW
3. Select appropriate mode
4. Set lung-protective ventilation parameters
5. Verify patient-ventilator interaction
6. Set appropriate alarm limits
7. Obtain baseline ABG
8. Document all settings

#### Critical Errors
- [ ] Using actual weight instead of ideal body weight
- [ ] Excessive tidal volume (>8 ml/kg)
- [ ] Not checking alarms
- [ ] Ignoring asynchrony

#### Debriefing Points
1. Lung-protective ventilation principles
2. Mode selection rationale
3. Alarm management
4. Patient-ventilator asynchrony recognition
5. Documentation requirements

### Scenario 2.3: Ventilator Weaning and Spontaneous Breathing Trial

**Level:** Intermediate  
**Duration:** 45 minutes  
**Setting:** ICU

#### Patient Profile
- **Name:** Patricia Brown
- **Age:** 62
- **Gender:** Female
- **Diagnosis:** Resolved pneumonia, ready for weaning assessment
- **Background:** Day 5 of mechanical ventilation, improving

#### Current Ventilator Settings
- Mode: SIMV
- VT: 450 ml
- Rate: 8
- PS: 10
- PEEP: 5
- FIO2: 40%

#### Weaning Parameters
- RSBI: 65
- NIF: -35 cmH2O
- Spontaneous VT: 400 ml
- ABG on current settings: pH 7.40, PaCO2 40, PaO2 85

#### Learning Objectives
1. Assess readiness for weaning
2. Perform spontaneous breathing trial
3. Evaluate SBT tolerance
4. Decide on extubation or return to support
5. Document weaning process

#### Scenario Flow

**Phase 1: Readiness Assessment (10 min)**
- Review weaning parameters
- Assess for readiness criteria
- Check underlying cause resolution

**Phase 2: SBT Initiation (10 min)**
- Explain procedure to patient
- Change to CPAP/PS or T-piece
- Set CPAP 5, PS 0 (or place on T-piece)
- Monitor closely

**Phase 3: Monitoring (15 min)**
- Monitor RR, HR, BP, SpO2
- Assess work of breathing
- Check for diaphoresis, anxiety
- Patient develops mild tachypnea (RR 28)

**Phase 4: Decision (10 min)**
- Evaluate tolerance
- Decide to continue, modify, or stop SBT
- Document findings and plan

#### SBT Tolerance Criteria
**Pass:**
- RR < 35
- SpO2 > 90%
- HR < 140 or <20% increase
- SBP < 180 or >90
- No diaphoresis
- No increased work of breathing
- No anxiety

**Fail:**
- RR > 35 or increase >50%
- SpO2 < 88%
- HR > 140 or increase >20%
- SBP > 180 or <90
- Diaphoresis
- Marked dyspnea
- Anxiety

#### Debriefing Points
1. Weaning readiness criteria
2. SBT methods (CPAP/PS vs T-piece)
3. Tolerance assessment
4. Extubation criteria
5. Failed SBT management

### Scenario 3.1: ARDS Management with Prone Positioning

**Level:** Advanced  
**Duration:** 60 minutes  
**Setting:** ICU

#### Patient Profile
- **Name:** James Anderson
- **Age:** 38
- **Gender:** Male
- **Diagnosis:** Severe ARDS secondary to sepsis
- **Background:** Day 3, deteriorating despite maximum conventional support

#### Current Status
- Mode: Volume A/C
- VT: 360 ml (6 ml/kg)
- Rate: 28
- PEEP: 12
- FIO2: 80%
- SpO2: 88%
- Plateau pressure: 32 cmH2O

#### Learning Objectives
1. Recognize severe ARDS requiring advanced interventions
2. Apply lung-protective ventilation strategy
3. Coordinate safe prone positioning
4. Manage ventilator during proning
5. Recognize complications

#### Equipment Needed
- Mechanical ventilator
- Prone positioning team (4-6 people)
- Pillows/positioning aids
- Pressure-relieving devices
- Emergency airway equipment
- Suction
- Arterial line for continuous BP monitoring

#### Scenario Flow

**Phase 1: Assessment (15 min)**
- Review Berlin criteria for ARDS
- Assess severity (PaO2/FIO2 ratio)
- Check contraindications to proning
- Discuss plan with team

**Phase 2: Preparation (10 min)**
- Pre-oxygenate
- Secure all lines/tubes
- Position pillows
- Assign team roles
- Review turning procedure

**Phase 3: Proning Procedure (15 min)**
- Execute safe turn to prone
- Reposition ET tube
- Adjust ventilator settings
- Verify tube position
- Check pressure points

**Phase 4: Management (15 min)**
- Monitor response (SpO2 improves to 94%)
- Manage secretions
- Check ventilator alarms
- Assess for complications
- Plan duration (16 hours)

#### Expected Actions
1. Confirm ARDS diagnosis and severity
2. Optimize lung-protective ventilation
3. Screen for proning contraindications
4. Coordinate multidisciplinary team
5. Execute safe proning procedure
6. Monitor for complications
7. Plan supination and reassessment

#### Contraindications to Proning
- Spinal instability
- Unstable fractures
- Open chest/abdominal wounds
- Severe hemodynamic instability
- ICP > 30 mmHg
- Pregnancy (relative)

#### Debriefing Points
1. ARDS definition and severity classification
2. Lung-protective ventilation principles
3. Proning indications and contraindications
4. Team coordination for safe proning
5. Complication recognition and management

### Scenario 3.2: Difficult Airway Management

**Level:** Advanced  
**Duration:** 45 minutes  
**Setting:** Emergency Department / ICU

#### Patient Profile
- **Name:** Michael Torres
- **Age:** 52
- **Gender:** Male
- **Diagnosis:** Facial trauma, suspected airway compromise
- **Background:** MVC, facial fractures, obese, limited mouth opening

#### Assessment Findings
- Mallampati IV
- Limited neck extension
- Facial swelling
- Obese (BMI 38)
- GCS 12
- SpO2 89% on NRB

#### Learning Objectives
1. Perform difficult airway assessment
2. Prepare difficult airway cart
3. Execute difficult airway algorithm
4. Use alternative airway devices
5. Perform surgical airway if needed

#### Equipment Needed
- Difficult airway cart
- Video laryngoscope
- Fiberoptic bronchoscope
- Supraglottic airways (LMA, King LT)
- Surgical airway kit
- BVM with PEEP valve
- High-flow nasal cannula
- Medications (induction, paralytics)

#### Scenario Flow

**Phase 1: Assessment (10 min)**
- Perform difficult airway assessment
- Predict intubation difficulty
- Assess bag-mask ventilation difficulty
- Develop airway management plan

**Phase 2: Preparation (10 min)**
- Position patient (ramp position)
- Pre-oxygenate (3 minutes)
- Prepare equipment
- Ensure backup plans ready

**Phase 3: Intubation Attempt (10 min)**
- First attempt with video laryngoscope
- Cormack-Lehane grade 3 view
- Failed intubation

**Phase 4: Rescue (10 min)**
- Maintain oxygenation with BVM
- Insert LMA successfully
- Ventilate through LMA
- Plan next steps

**Phase 5: Surgical Airway (5 min)**
- If LMA fails, proceed to surgical cricothyrotomy
- Demonstrate technique (simulated)

#### Expected Actions
1. Recognize difficult airway
2. Call for appropriate help
3. Position optimally
4. Pre-oxygenate adequately
5. Have backup plans ready
6. Limit intubation attempts
7. Maintain oxygenation priority
8. Execute rescue techniques

#### Difficult Airway Algorithm
1. Pre-oxygenation
2. First attempt (best technique/operator)
3. If fail: Face mask ventilation
4. If FMV adequate: Second attempt
5. If FMV inadequate: Supraglottic airway
6. If SGA adequate: Second attempt or surgical airway
7. If SGA inadequate: Surgical airway

#### Debriefing Points
1. Difficult airway prediction
2. Preparation importance
3. Algorithm application
4. Rescue techniques
5. Human factors in airway emergencies

## Scenario Development Guide

### Creating New Scenarios

1. **Define Learning Objectives**
   - What should participants learn?
   - What competencies should be demonstrated?

2. **Develop Patient Profile**
   - Create realistic demographics
   - Include relevant history
   - Design appropriate vital signs

3. **Structure the Flow**
   - Plan phases with clear triggers
   - Build in decision points
   - Include expected actions

4. **Design Confounding Factors**
   - Add realistic complications
   - Include distractors
   - Test critical thinking

5. **Create Assessment Tools**
   - Competency checklists
   - Debriefing guides
   - Evaluation rubrics

6. **Pilot Test**
   - Run with experienced staff
   - Refine timing
   - Adjust difficulty

### Scenario Library Maintenance

- Review scenarios annually
- Update based on new evidence
- Incorporate participant feedback
- Add new equipment/protocols
- Retire outdated scenarios
