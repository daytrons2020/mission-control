# Medical Images for Respiratory Education

## Image Sources

These images are sourced from public domain and Creative Commons sources for educational use.

### Recommended Sources for Medical Images:
1. **CDC Public Health Image Library (PHIL)** - https://phil.cdc.gov/
2. **NIH Visuals Online** - https://visualsonline.cancer.gov/
3. **Wikimedia Commons** - https://commons.wikimedia.org/
4. **Radiopaedia** - https://radiopaedia.org/ (CC BY-NC-SA)
5. **AnatomyZone** - https://www.anatomyzone.com/

### Usage Notes:
- All images should be properly attributed
- Check license terms before public distribution
- For internal educational use only

## Manual Download Instructions

Due to API restrictions and rate limiting, some images may need to be manually downloaded:

1. Visit the source websites above
2. Search for relevant medical images
3. Download high-resolution versions
4. Save to this folder with descriptive filenames
5. Update PowerPoints to reference new images
