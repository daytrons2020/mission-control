### Suno AI for Music Generation

**Best Prompts by Channel:**

**Lofi Music:**
```
"lofi hip hop beat, chill study music, soft piano melody, 
jazz chords, dusty vinyl crackle, relaxed tempo 75 BPM, 
cozy atmosphere, no vocals"
```

**Kids Sleep Music:**
```
"gentle lullaby, soft piano, music box tones, 
dreamy atmosphere, 55 BPM, bedtime music for children, 
calming, no percussion, ambient strings"
```

**Ambient Sleep:**
```
"brown noise texture, deep rumble, soothing, 
continuous drone, no melody, sleep aid, 
10 hours seamless loop"
```

### ElevenLabs Voice Settings

**Best Voices by Channel:**

**Kids Education:**
- Voice: "Bella" or "Rachel"
- Stability: 0.5 (more expressive)
- Clarity + Similarity: 0.75
- Style: 0.3 (slight emphasis)

**Kids Stories:**
- Voice: "Bella" (warm, storytelling)
- Stability: 0.6
- Clarity + Similarity: 0.8
- Style: 0.4 (expressive for characters)

**General Narration:**
- Voice: "Adam" or "Antoni"
- Stability: 0.7
- Clarity + Similarity: 0.75
- Style: 0.2

## API Documentation

### YouTube Data API
- **Docs:** https://developers.google.com/youtube/v3
- **Quota:** 10,000 units/day (free)
- **Key Endpoints:**
  - `videos.insert` - Upload videos
  - `videos.update` - Update metadata
  - `search.list` - Search videos/channels
  - `channels.list` - Get channel stats

### OpenAI API
- **Docs:** https://platform.openai.com/docs
- **Pricing:** Pay per token
- **Models:** GPT-4, GPT-3.5, DALL-E 3

### ElevenLabs API
- **Docs:** https://docs.elevenlabs.io
- **Pricing:** Characters per month
- **Features:** Voice cloning, TTS, STS
