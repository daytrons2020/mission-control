# Ambient Sleep - Thumbnail Templates

## Specifications
- **Size:** 1280 x 720 pixels
- **Format:** JPG (dark images compress well)
- **Style:** Dark, peaceful, minimal text

---

## Template 1: "Night Landscape"

### Layout
```
+------------------------------------------+
|                                          |
|     [Moon/Stars]                         |
|                                          |
|  [Dark Landscape - Forest/Ocean/Mtn]     |
|                                          |
|         Sleep Sounds                     |
+------------------------------------------+
```

### Design Elements
- **Sky:** Deep gradient, stars, moon
- **Landscape:** Silhouette or low-lit nature
- **Mood:** Serene, mysterious, calming
- **Text:** Minimal, very subtle
- **Colors:** Deep blues, purples, blacks

## Post-Processing Tips

### Darken for Mood
1. Open in GIMP
2. Colors → Curves
3. Slightly pull down midtones
4. Keep some highlight detail

### Add Star Glow
1. Create new layer
2. Paint white dots for stars
3. Filters → Blur → Gaussian Blur (2-3px)
4. Set blend mode to Screen
5. Duplicate for brighter stars

### Smooth Gradients
1. Use gradient tool for backgrounds
2. Colors → Blur → Selective Gaussian Blur
3. This prevents banding in dark areas

## Important Notes

1. **Text Minimalism:** Sleep thumbnails should have almost no text
2. **Dark Colors:** Embrace the darkness - it's calming
3. **Consistency:** Use similar mood across all thumbnails
4. **Testing:** Check how it looks in YouTube's dark mode
5. **Compression:** Dark images can use higher compression
