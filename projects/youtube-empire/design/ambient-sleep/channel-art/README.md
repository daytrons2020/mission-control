# Ambient Sleep - Channel Art

## Specifications
- **Size:** 2560 x 1440 pixels
- **Safe Area:** 1546 x 423 pixels (center)
- **Format:** JPG (smaller file size for dark images)
- **Style:** Dark, serene, minimal

---

## Design Concept: "Night Sky Sanctuary"

### Layout
```
+----------------------------------------------------------+
|  [Stars scattered]                                       |
|                    [Moon]                                |
|                                                          |
|  +--------------------------------------------------+    |
|  |                                                  |    |
|  |              SAFE ZONE - 1546x423                |    |
|  |                                                  |    |
|  |         CHANNEL NAME (subtle)                    |    |
|  |         "Sleep peacefully"                       |    |
|  |                                                  |    |
|  +--------------------------------------------------+    |
|                                                          |
|  [Silhouette landscape - trees, mountains, or ocean]     |
+----------------------------------------------------------+
```

### Elements
1. **Background:** Deep gradient #1A1A2E to #16213E
2. **Stars:** Scattered across top
3. **Moon:** Soft glow, positioned above safe zone
4. **Landscape:** Silhouette at bottom
5. **Text:** Very subtle, minimal

---

## Color Zones

| Zone | Color | Purpose |
|------|-------|---------|
| Top | #1A1A2E (Midnight) | Deep space |
| Middle | #16213E (Deep Purple) | Transition |
| Bottom | #0F3460 (Forest Shadow) | Landscape |
| Accents | #A8A4CE (Lavender) | Moon, stars |

---

## GIMP Creation Steps

1. **New Image**
   - File → New → 2560 x 1440
   - Fill with #1A1A2E

2. **Create Gradient**
   - Gradient tool
   - Foreground: #1A1A2E
   - Background: #16213E
   - Drag from top to bottom

3. **Add Stars**
   - New layer
   - Paint brush, white, various sizes
   - Paint scattered dots
   - Gaussian blur: 1px for glow

4. **Add Moon**
   - Ellipse select tool
   - Fill with #A8A4CE
   - Gaussian blur: 20px
   - Reduce opacity to 70%

5. **Add Landscape**
   - New layer at bottom
   - Paint silhouette with #0F3460
   - Trees, mountains, or hills

6. **Add Text (Optional)**
   - Keep very subtle
   - Cormorant Garamond, light weight
   - Color: #A8A4CE at 50% opacity
