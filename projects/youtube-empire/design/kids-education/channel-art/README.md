## Canva Free Creation Steps

1. **Create New Design**
   - Go to canva.com
   - Custom size: 2560 x 1440 pixels
   - Name: "Kids Education Channel Art"

2. **Add Background**
   - Elements → Gradient
   - Set colors: #FFD93D to #4ECDC4
   - Angle: 135 degrees

3. **Add Floating Elements**
   - Search "education" in Elements
   - Add: ABC, 123, shapes, stars
   - Position around edges
   - Resize: 100-200px

4. **Add Text**
   - Heading: Channel name
   - Font: Fredoka One
   - Size: 120px (adjust for length)
   - Color: #2D3436 (Charcoal)
   - Position: Center of safe zone

5. **Add Tagline**
   - Text: "Learn • Play • Grow"
   - Font: Nunito
   - Size: 48px
   - Position: Below channel name

6. **Download**
   - Format: PNG (better quality)
   - Use for YouTube channel

## Safe Zone Template

Download or create a safe zone overlay:
- Rectangle: 1546 x 423 pixels
- Centered on 2560 x 1440 canvas
- Use while designing, remove before export

This ensures all important content is visible on all devices.

---

## Mobile Preview Check

YouTube channel art displays differently on devices:

- **Desktop:** Full 2560 x 1440 visible
- **TV:** Full image visible
- **Tablet:** Slightly cropped
- **Mobile:** Only center 1546 x 423 visible

**Always design for mobile first** - put critical info in safe zone.
