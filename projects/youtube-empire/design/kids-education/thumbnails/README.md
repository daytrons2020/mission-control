# Kids Education - Thumbnail Templates

## Specifications
- **Size:** 1280 x 720 pixels
- **Format:** PNG (for transparency) or JPG
- **Max File Size:** 2MB

---

## Template 1: "Big Number/Concept"

### Layout
```
+------------------------------------------+
|                                          |
|     [Large Number/Concept - 40% width]   |
|                                          |
|  [Mascot]      LEARNING TITLE            |
|              Simple Subtitle             |
+------------------------------------------+
```

### Design Elements
- **Background:** Gradient from #FFD93D (top) to #4ECDC4 (bottom)
- **Main Element:** Large number or concept icon (40% of thumbnail)
- **Character:** Friendly mascot in bottom-left corner
- **Title:** Fredoka One font, 72-96px, white with dark outline
- **Subtitle:** Nunito font, 36-48px
- **Accents:** Stars, confetti, or sparkles

### Canva Free Creation Steps
1. Go to canva.com → Create design → YouTube Thumbnail (1280x720)
2. Background: Gradient, colors #FFD93D to #4ECDC4, 45-degree angle
3. Add Elements → Shapes → Rounded rectangle for title background
4. Text: Add heading "Fredoka One" font, size 72
5. Add free illustrations from Elements (search "kids", "education")
6. Download as PNG

## Template 2: "Before/After Learning"

### Layout
```
+------------------------------------------+
|  BEFORE        →→→         AFTER         |
|  [Question]    Arrow      [Answer]       |
|                                          |
|         "Learn With Us!"                 |
+------------------------------------------+
```

### Design Elements
- **Split:** 50/50 vertical split
- **Left Side:** Question mark or problem, lighter color
- **Right Side:** Answer or solution, brighter color
- **Divider:** Arrow or transformation visual
- **Text:** "Learn" badge at bottom

### GIMP Creation Steps
1. Open GIMP → File → New → 1280x720
2. Create two layers for left/right sides
3. Fill left with #95E1D3, right with #FFD93D
4. Add gradient divider in middle
5. Add text layers with Nunito font
6. Export as PNG

### Design Elements
- **Character:** Takes left half, expressive, eye-catching

## Color Usage Guide

| Element | Color | Usage |
|---------|-------|-------|
| Primary Background | #FFD93D | Main backgrounds, highlights |
| Secondary Background | #4ECDC4 | Alternate sections, trust |
| Accent/CTA | #FF6B6B | Buttons, important text |
| Success/Positive | #95E1D3 | Correct answers, progress |
| Text | #2D3436 | All text, outlines |
| Light Background | #FFF9F0 | Clean areas, cards |
