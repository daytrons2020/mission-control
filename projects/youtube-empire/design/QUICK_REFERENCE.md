## Thumbnail Specs (All Channels)
- **Size:** 1280 x 720 pixels
- **Format:** PNG or JPG
- **Max Size:** 2MB
- **Text Safe Zone:** Center 1238 x 165 pixels

## Channel Art Specs (All Channels)
- **Size:** 2560 x 1440 pixels
- **Safe Zone:** 1546 x 423 pixels (center)
- **Format:** PNG or JPG
- **Max Size:** 6MB

---

## Quick Creation Workflow

### For Each Thumbnail:
1. Generate background image (Bing Image Creator)
2. Upload to Canva Free
3. Add text with channel font
4. Download as PNG
5. Compress if needed (tinypng.com)

### Time Estimate:
- **First thumbnail:** 30 minutes
- **After template:** 5-10 minutes each

## Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Image too big | Use tinypng.com to compress |
| Text hard to read | Add outline or shadow |
| Colors look different | Check monitor calibration |
| Can't remove background | Use remove.bg (free tier) |
| Need inspiration | Check similar YouTube channels |
