# Lofi Music - Thumbnail Templates

## Specifications
- **Size:** 1280 x 720 pixels
- **Format:** JPG (smaller file size) or PNG
- **Style:** Muted, vintage, film grain aesthetic

---

## Template 1: "Study Scene"

### Layout
```
+------------------------------------------+
|  [Window with rain/sunset view]          |
|                                          |
|     [Character studying]                 |
|     [Desk with books, plant, coffee]     |
|                                          |
|  Lo-Fi Study Session    [Vinyl icon]     |
+------------------------------------------+
```

### Design Elements
- **Background:** Window view with atmospheric weather
- **Character:** Anime-style, focused on studying
- **Desk Items:** Books, laptop, plant, coffee cup, headphones
- **Lighting:** Warm lamp light, sunset glow
- **Text:** Minimal, bottom-right corner
- **Effects:** Film grain overlay, slight vignette

### Color Palette
| Element | Color | Usage |
|---------|-------|-------|
| Primary | #D4A5A5 | Warm accents, character tones |
| Secondary | #9CAF88 | Plants, nature elements |
| Background | #F5F1E8 | Walls, paper |
| Text | #8B7355 | All text elements |
| Dark | #3D3D3D | Shadows, outlines |

### Canva Free Creation Steps
1. Use AI-generated background (Bing Image Creator)
2. Upload to Canva as base image
3. Add text with Josefin Sans font
4. Download free film grain texture
5. Overlay grain in GIMP if needed

## Post-Processing (GIMP - Free)

### Film Grain Effect
1. Open image in GIMP
2. Filters → Noise → RGB Noise
3. Set to low value (0.02-0.05)
4. Blend mode: Overlay at 30% opacity

### Vignette Effect
1. Create new layer
2. Fill with black
3. Add layer mask
4. Gradient tool: radial from center (black to transparent)
5. Reduce layer opacity to 20-30%

### Color Grading
1. Colors → Color Balance
2. Shadows: +10 red, -5 blue
3. Midtones: +5 yellow
4. Highlights: slight warm tint
