# Lofi Music - Channel Art

## Specifications
- **Size:** 2560 x 1440 pixels
- **Safe Area:** 1546 x 423 pixels (center)
- **Format:** JPG or PNG
- **Style:** Muted, aesthetic, cozy

---

## Design Concept: "Cozy Study Sanctuary"

### Layout
```
+----------------------------------------------------------+
|  [Vinyl record]                                          |
|                    LO-FI CHANNEL NAME                    |
|              "Music for studying, relaxing"              |
|                                                          |
|  +--------------------------------------------------+    |
|  |                                                  |    |
|  |              SAFE ZONE - 1546x423                |    |
|  |                                                  |    |
|  |         ALL IMPORTANT CONTENT HERE               |    |
|  |                                                  |    |
|  +--------------------------------------------------+    |
|                                                          |
|  [Cozy room illustration fades to edges]                 |
+----------------------------------------------------------+
```

### Elements
1. **Background:** Soft gradient or cozy room illustration
2. **Vinyl Records:** Floating on sides (decorative)
3. **Channel Name:** Josefin Sans, centered
4. **Tagline:** Descriptive subtitle
5. **Atmosphere:** Warm, inviting, study-friendly

## Canva Free Creation Steps

1. **Create Design**
   - Size: 2560 x 1440
   - Background: #F5F1E8

2. **Add Illustration**
   - Use unDraw.co illustration
   - Customize colors to match palette
   - Upload to Canva
   - Position as background element

3. **Add Vinyl Elements**
   - Search "vinyl record" in Elements
   - Position on sides
   - Add rotation for dynamic feel

4. **Add Text**
   - Channel name: Josefin Sans, 100px
   - Tagline: Inter, 40px
   - Center in safe zone

5. **Add Film Grain**
   - Download free grain texture
   - Upload and overlay at 20%
