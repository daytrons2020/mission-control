## Canva Free Creation Steps

1. **Create Design**
   - Size: 2560 x 1440
   - Background: #FDF6E3

2. **Add Paper Texture**
   - Search "paper texture" in Elements
   - Or upload free texture
   - Overlay at 30%

3. **Add Floating Pages**
   - Search "book page" or "paper"
   - Position around edges
   - Rotate for dynamic feel
   - Add drop shadows

4. **Add Sparkles**
   - Search "sparkle" or "star"
   - Scatter around channel name
   - Gold color #F1C40F

5. **Add Text**
   - Channel name: Amatic SC, 120px
   - Tagline: Poppins, 40px
   - Color: #5D4E37

6. **Add Characters**
   - Use free illustrations
   - Position peeking from edges
   - Partially outside frame
