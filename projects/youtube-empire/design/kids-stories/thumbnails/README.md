# Kids Stories - Thumbnail Templates

## Specifications
- **Size:** 1280 x 720 pixels
- **Format:** PNG (for paper textures) or JPG
- **Style:** Whimsical, storybook, hand-drawn aesthetic

---

## Template 1: "Storybook Cover"

### Layout
```
+------------------------------------------+
|  [Decorative Border]                     |
|                                          |
|     [Illustrated Scene]                  |
|     [Characters in action]               |
|                                          |
|     "Story Title Here"                   |
|     [Read Aloud badge]                   |
+------------------------------------------+
```

### Design Elements
- **Border:** Decorative storybook frame
- **Illustration:** Scene from the story
- **Title:** Storybook-style font
- **Badge:** "Read Aloud" or duration indicator
- **Texture:** Paper texture overlay

## Template 2: "Character Portrait"

### Layout
```
+------------------------------------------+
|                                          |
|  [Main Character - Large, Expressive]    |
|                                          |
|     Story Title                          |
|     [Small scene preview]                |
+------------------------------------------+
```

### Design Elements
- **Character:** Takes 50%+ of frame
- **Expression:** Happy, curious, or surprised
- **Title:** Below or beside character
- **Preview:** Small inset showing scene
- **Background:** Simple, doesn't compete

## Post-Processing Tips

### Paper Texture Overlay
1. Find free paper texture (beige/cream)
2. Open in GIMP as new layer
3. Set blend mode to Multiply at 20-30%
4. This gives authentic storybook feel

### Watercolor Effect
1. Duplicate image layer
2. Filters → Artistic → Watercolor (subtle)
3. Blend at 30-50% opacity
4. Or use overlay textures

### Hand-drawn Outline
1. Duplicate layer
2. Filters → Edge-Detect → Difference of Gaussians
3. Invert colors
4. Set to Multiply blend mode
5. Reduce opacity to 10-20%
