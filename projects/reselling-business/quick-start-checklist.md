# Reselling Business - Quick Start Checklist

## Pre-Launch Setup

### Business Foundation
- [ ] Choose business structure (LLC recommended)
- [ ] Obtain EIN (Employer Identification Number)
- [ ] Get resale certificate/tax ID
- [ ] Open business bank account
- [ ] Set up accounting software (QuickBooks, Xero, or Wave)

### Platform Registration
- [ ] Amazon Professional Seller Account ($39.99/mo)
  - Complete identity verification
  - Set up tax information
  - Configure deposit method
- [ ] eBay Seller Account
  - Upgrade to Store subscription if needed
  - Set up managed payments
  - Configure shipping preferences

### Tool Subscriptions (Starter Stack)
- [ ] Jungle Scout or Helium 10 (product research)
- [ ] Keepa (price tracking)
- [ ] SellerAmp (profit calculator)
- [ ] 3Dsellers or InkFrog (eBay listing)
- [ ] InventoryLab (Amazon workflow)

## Week 1-2: First Products

### Research Phase
- [ ] Identify 3-5 potential product categories
- [ ] Analyze 20+ products using research tools
- [ ] Validate profitability with profit calculator
- [ ] Check for restrictions/gated categories
- [ ] Verify sourcing availability

### Sourcing Phase
- [ ] Visit 2-3 retail stores for arbitrage opportunities
- [ ] Set up alerts on 3+ online deal sites
- [ ] Contact 1-2 wholesale suppliers
- [ ] Purchase first test batch (5-10 units)

### Listing Phase
- [ ] Create optimized Amazon listings
- [ ] Create eBay listings with templates
- [ ] Set competitive pricing
- [ ] Enable promoted listings (start with 5%)

## Week 3-4: Automation Setup

### Price Monitoring
- [ ] Configure Keepa alerts for 20+ target products
- [ ] Set up deal alerts on Slickdeals, Brad's Deals
- [ ] Create spreadsheet for tracking opportunities
- [ ] Test alert notification system

### Listing Automation
- [ ] Create listing templates for eBay
- [ ] Set up bulk listing workflow
- [ ] Configure inventory sync (if multi-channel)
- [ ] Schedule listing times for optimal visibility

### Profit Tracking
- [ ] Set up profit tracking dashboard
- [ ] Document all costs (product, fees, shipping)
- [ ] Create weekly profit review process
- [ ] Set minimum margin thresholds

## Month 2: Scaling

### Inventory Expansion
- [ ] Increase active SKUs to 25-50
- [ ] Add second sourcing method
- [ ] Test wholesale or liquidation lots
- [ ] Optimize based on first month's data

### Process Optimization
- [ ] Document all standard operating procedures
- [ ] Create templates for common tasks
- [ ] Set up automated repricing rules
- [ ] Implement customer service templates

### Analysis
- [ ] Review profit/loss by product
- [ ] Identify top 10% performers
- [ ] Cut underperforming products
- [ ] Adjust sourcing strategy

## Month 3+: 24/7 Operation

### Full Automation
- [ ] All price monitoring automated
- [ ] Auto-purchase triggers for known products
- [ ] Inventory management on autopilot
- [ ] Customer service response templates

### Growth Activities
- [ ] Expand to 100+ SKUs
- [ ] Consider prep center for FBA
- [ ] Evaluate virtual assistant for tasks
- [ ] Add additional sales channels (Walmart, etc.)

### Weekly Routine (Minimal)
- [ ] Review profit dashboard (15 min)
- [ ] Check low inventory alerts (10 min)
- [ ] Respond to customer messages (15 min)
- [ ] Review and approve new purchases (20 min)

---

## Key Metrics to Track

### Daily
- Sales volume and revenue
- New arbitrage opportunities found
- Inventory levels

### Weekly
- Profit by product/SKU
- Return/refund rates
- PPC performance (if applicable)
- Sourcing pipeline

### Monthly
- Overall profit/loss
- ROI by sourcing method
- Cash flow analysis
- Business growth metrics

---

## Red Flags to Watch

### Stop Immediately If:
- [ ] Account health metrics declining
- [ ] Multiple customer complaints
- [ ] Cash flow negative for 2+ weeks
- [ ] Suspicious supplier activity
- [ ] Product authenticity questions

### Review Strategy If:
- [ ] Margins consistently below 20%
- [ ] Inventory not selling after 60 days
- [ ] Competition increasing rapidly
- [ ] Fees eating into profits
