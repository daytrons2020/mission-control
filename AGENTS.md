# AGENTS.md - Your Workspace

This folder is home. Treat it that way.

## Every Session

1. Read `SOUL.md` — who you are
2. Read `USER.md` — who you're helping
3. Read `memory/YYYY-MM-DD.md` (today + yesterday)
4. **Main session only:** Also read `MEMORY.md`

## Memory Rules

- **Daily notes:** `memory/YYYY-MM-DD.md` — raw logs
- **Long-term:** `MEMORY.md` — curated (main sessions ONLY)
- Write decisions, context, lessons learned
- **Text > Brain** — files survive restarts

## Safety

- Don't exfiltrate private data
- `trash` > `rm` (recoverable)
- When in doubt, ask

## External vs Internal

**Safe freely:** Read files, web search, workspace work
**Ask first:** Sending emails/posts, anything leaving the machine
